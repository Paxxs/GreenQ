

绿色版新增的“一键切换组件”功能，现已完美实现了所有安装版的选项切换功能。
使用方法：直接双击目录下“一键切换组件.bat”，然后按提示操作即可。
感谢网友“喵星人”制作的脚本加持！


基于官方数字签名为2019年3月7日的 QQ 9.1.0 (24712) 正式版制作


注意：所有的修改版都有被锁的风险，是否使用自己斟酌，本人不承担任何责任！
注意：安装使用QQ前请先自备安装 Visual C++ 2017 和 Universal C 运行库，QQ 运行必备
注意：安装使用QQ前请先自备安装 Adobe Flash Player 最新版组件，QQ 秀正常显示必备 (Win10 系统在线更新即可)
注意：默认整合的 NtrQQ 可能会被一些杀软误报和拦截，至于怎么做我就不多说了
      NtrQQ 里也关闭了主面板最小高宽的选项，想要缩的更窄更小的手动去 NtrQQ.ini 里开启：KillMainPanelSize=1
      NtrQQ 里关闭了一个默认开启且可能会导致自动离线的选项：KillQQKickOut=0，本身已经去校验：KillQQKickOut=0
      更多 NtrQQ 选项设置可以自行翻阅 NtrQQ.ini，Lance.Moe 都有详细说明

提示：请尽量使用管理员权限安装，外加关闭 UAC，可解各种疑难杂症
提示：请尽量不要覆盖安装，每次更新前先卸载之前的版本再安装新版
提示：安装时如有提示文件占用，请手动找下并结束 QQ.exe、QQExternal.exe、TXPlatform.exe 等相关进程一般就能重试继续，再不行就看看其他腾讯软件都开了些啥，比如：腾讯电脑管家、小Q书桌，都关了再安装试试
提示：支持覆盖安装时切换选项，如系统声音、托盘图标、移动设备图标等多个选项之间来回切换
提示：选中QQ收藏・Outlook助手安装前先关闭 Outlook，否则助手会安装不能
